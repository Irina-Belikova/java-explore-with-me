# java-explore-with-me
Template repository for ExploreWithMe project.

## Краткое описание feature-comments.
1. Оставлять комментарии к событию могут любые зарегистрированные пользователи, включая автора события.
2. Комментарий можно оставить только к опубликованному событию.
3. Оставлять комментарий можно независимо от даты события (например, можно задать вопрос по предстоящему мероприятию).
4. Можно ответить на чей-то комментарий.
5. Пользователь может удалить свой комментарий из базы данных.
6. Администратор может скрыть некорректный комментарий (canceled, но не удаление из БД). При этом ответы на комментарий тоже скрываются.
7. Пользователь может получить ответы на свой комментарий.
8. Просматривать комментарии к событию могут только зарегистрированные пользователи.
